### Changes by the mooR authors

- Converted to mooR objdef structure
- Switched to `argon2` for password crypt
- Changes throughout to avoid assignment into type literals (which is a compile error in mooR but not LambdaMOO)
- Changes throughout to better support mooR UUID-style object references
- New $login.welcome_message, with djot/markdown style content
- Added `$oauth_challenge` `$ouath_create` `$oauth_connect`
